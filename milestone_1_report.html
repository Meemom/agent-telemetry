Milestone 1 Scope

Milestone 1 establishes the foundation for AgentTelemetry: a Python package scaffold, CLI entrypoint, core Pydantic schemas, JSONL trace writer, and a basic test suite. This creates the base needed for later LangGraph scanning, runtime instrumentation, adversarial testing, and reporting.

Package Scaffold Created a src/agenttelemetry Python package.
CLI Added Typer commands for version output and trace initialization.
Schemas Added Pydantic models for graphs, traces, findings, and tests.
Trace Writer Added JSONL trace persistence and an in-memory collector.

CLI Behavior

The package exposes the agenttelemetry command.

agenttelemetry version
agenttelemetry init-trace --workflow-name demo --output runs/trace.jsonl

The init-trace command writes a valid JSONL runtime event, proving the CLI, trace model, and writer are connected.

Core Schemas

  • WorkflowGraph, GraphNode, GraphEdge, and ToolDefinition
  • RuntimeEvent, RuntimeEventType, and TraceRun
  • SecurityFinding, Severity, and Confidence
  • AssertionResult and TestResult

Notes

  • The first install attempt revealed a Hatch package discovery issue caused by the hyphenated project name. This was fixed by explicitly configuring packages = ["src/agenttelemetry"].
  • Runtime LangGraph instrumentation is intentionally not implemented in Milestone 1. The current trace writer and schemas are the foundation for that next step.
  • Generated local artifacts such as .venv and .pytest_cache are ignored by the repository.
